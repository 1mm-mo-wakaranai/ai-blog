## この記事で解決すること

「何かを勉強したいけど、何から始めればいいか分からない」

ChatGPTに自分の状況を伝えるだけで、あなた専用の学習プランを作ってくれます。

## なぜChatGPTに学習プランを作ってもらうのか

- 本やWebサイトの学習ロードマップは「万人向け」
- ChatGPTなら「あなたの状況」に合わせたプランを作れる
- 途中で変更や調整もできる
- 無料で何度でも相談できる

ChatGPTの基本的な使い方がまだ分からない方は、まず[ChatGPTの始め方](/posts/chatgpt-first-step/)を読んでみてください。

## コピペで使えるプロンプト

以下をChatGPTにコピペして、【】の中を自分の情報に書き換えてください。

```
以下の条件で、私専用の学習プランを作ってください。

■ 学びたいこと
【例：Pythonプログラミング】

■ 現在のレベル
【例：完全な初心者。プログラミング経験ゼロ】

■ 学習の目的
【例：3ヶ月後に簡単なWebアプリを作れるようになりたい】

■ 使える時間
【例：平日は1日1時間、土日は3時間ずつ】

■ 学習スタイルの好み
【例：動画より本で学ぶ方が好き。手を動かしながら覚えたい】

■ 出力フォーマット
- 週ごとのスケジュール（12週間分）
- 各週の学習テーマと具体的なやること
- おすすめの教材（無料のものを優先）
- 各週の到達目標（何ができるようになるか）
```

## 実際の使用例

```
以下の条件で、私専用の学習プランを作ってください。

完全な初心者。Excelは使える。

- 各週の学習テーマと具体的なやること
- おすすめの無料教材
- 各週の到達目標
```

## 学習プランを作った後の活用法

### 進捗を報告する

毎週、ChatGPTに進捗を報告すると、プランを調整してくれます。進捗管理をもっと体系的にやりたい方は、[ChatGPTでTODOリスト・タスク管理をする方法](/posts/chatgpt-todo-management/)もあわせてご覧ください。

```
先週の学習プランの進捗報告です。
- 第2週のテーマ「関数の基礎」は完了しました
- ただし、引数の概念がまだよく分かりません
- 来週は予定より時間が取れそうです（土日5時間ずつ）

プランを調整してください。
```

なお、毎回「自分は初心者で〜」と伝えるのが面倒な場合は、[ChatGPTカスタム指示の設定方法](/posts/chatgpt-custom-instructions/)で事前に登録しておくと便利です。

### つまずいたら相談する

```
学習プランの第3週で「クラス」の概念が理解できません。
もっと簡単な例で説明してもらえますか？
必要なら、第3週のスケジュールを延長してください。
```

## よくある質問（FAQ）

### Q: 学習プランはどのくらいの期間で作ってもらえますか？
A: 数秒〜十数秒で生成されます。1週間〜半年程度のプランが実用的です。期間が長すぎると後半が大まかになるので、3ヶ月ごとに区切って作り直すのがおすすめです。

### Q: ChatGPTが提案する教材は信頼できますか？
A: 有名な教材やサービスを提案してくれることが多いですが、リンク切れや情報が古い場合もあります。教材名で検索して、実際に存在するか・最新版かを確認してから使ってください。

### Q: 途中でやる気がなくなったらどうすればいいですか？
A: ChatGPTに「モチベーションが下がっています。ペースを落としたプランに変更してください」と伝えてみてください。学習量を減らしたり、楽しめる内容を増やしたりしたプランを提案してくれます。

### Q: ChatGPT以外のAIでも学習プランは作れますか？
A: はい、ClaudeやGeminiでも同じプロンプトが使えます。AIごとの特徴については[ChatGPTとGemini、結局どっちがいい？](/posts/gemini-vs-chatgpt/)や[Claudeの特徴とChatGPTとの違い](/posts/claude-what-is-it/)を参考にしてください。
