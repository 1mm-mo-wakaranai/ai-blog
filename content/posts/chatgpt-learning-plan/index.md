## この記事で解決すること

「何かを勉強したいけど、何から始めればいいか分からない」

ChatGPTに自分の状況を伝えるだけで、あなた専用の学習プランを作ってくれます。

## なぜChatGPTに学習プランを作ってもらうのか

- 本やWebサイトの学習ロードマップは「万人向け」
- ChatGPTなら「あなたの状況」に合わせたプランを作れる
- 途中で変更や調整もできる
- 無料で何度でも相談できる

## コピペで使えるプロンプト

以下をChatGPTにコピペして、【】の中を自分の情報に書き換えてください。

```
以下の条件で、私専用の学習プランを作ってください。

■ 学びたいこと
【例：Pythonプログラミング】

■ 現在のレベル
【例：完全な初心者。プログラミング経験ゼロ】

■ 学習の目的
【例：3ヶ月後に簡単なWebアプリを作れるようになりたい】

■ 使える時間
【例：平日は1日1時間、土日は3時間ずつ】

■ 学習スタイルの好み
【例：動画より本で学ぶ方が好き。手を動かしながら覚えたい】

■ 出力フォーマット
- 週ごとのスケジュール（12週間分）
- 各週の学習テーマと具体的なやること
- おすすめの教材（無料のものを優先）
- 各週の到達目標（何ができるようになるか）
```

## 実際の使用例

### 例1：プログラミング学習

```
以下の条件で、私専用の学習プランを作ってください。

■ 学びたいこと
Pythonプログラミング

■ 現在のレベル
完全な初心者。Excelは使える。

■ 学習の目的
仕事で使うデータ処理を自動化したい

■ 使える時間
平日30分、土日2時間ずつ

■ 学習スタイルの好み
手を動かしながら覚えたい。長い動画は苦手。

■ 出力フォーマット
- 8週間のスケジュール
- 各週の学習テーマと具体的なやること
- おすすめの無料教材
- 各週の到達目標
```

### 例2：英語学習

```
以下の条件で、私専用の英語学習プランを作ってください。

■ 学びたいこと
ビジネス英語（メール・会議で使えるレベル）

■ 現在のレベル
TOEIC 500点程度。読み書きはなんとかできるが、リスニングとスピーキングが苦手。

■ 学習の目的
半年後に海外チームとの会議に参加できるようになりたい

■ 使える時間
通勤時間30分（片道）+ 寝る前30分

■ 学習スタイルの好み
アプリやポッドキャストを使いたい。机に向かう勉強は続かない。

■ 出力フォーマット
- 月ごとのスケジュール（6ヶ月分）
- 各月の重点テーマ
- おすすめのアプリ・教材
- 各月の到達目標
```

### 例3：資格試験

```
以下の条件で、私専用の資格試験対策プランを作ってください。

■ 学びたいこと
基本情報技術者試験の合格

■ 現在のレベル
IT業界未経験。文系出身。

■ 学習の目的
4月の試験に合格したい（残り3ヶ月）

■ 使える時間
平日1時間、土日4時間ずつ

■ 学習スタイルの好み
過去問を解きながら覚えるタイプ

■ 出力フォーマット
- 12週間のスケジュール
- 各週の学習範囲
- おすすめの参考書・Webサイト
- 模擬試験のタイミング
```

## 学習プランを作った後の活用法

### 進捗を報告する

毎週、ChatGPTに進捗を報告すると、プランを調整してくれます。

```
先週の学習プランの進捗報告です。
- 第2週のテーマ「関数の基礎」は完了しました
- ただし、引数の概念がまだよく分かりません
- 来週は予定より時間が取れそうです（土日5時間ずつ）

プランを調整してください。
```

### つまずいたら相談する

```
学習プランの第3週で「クラス」の概念が理解できません。
もっと簡単な例で説明してもらえますか？
必要なら、第3週のスケジュールを延長してください。
```

## まとめ

- ChatGPTに状況を伝えるだけで、オーダーメイドの学習プランが作れる
- プロンプトの【】を自分の情報に書き換えるだけ
- 進捗報告をすれば、プランを随時調整してくれる
- プログラミング、英語、資格試験、何でも対応
