## この記事で解決すること

「やることが多すぎて、何から手をつければいいか分からない」

ChatGPTにタスクを整理してもらえば、優先順位が明確になり、抜け漏れも防げます。ChatGPTの基本的な使い方については、[ChatGPTの始め方](/posts/chatgpt-first-step/)をご覧ください。

## ChatGPTでタスク管理するメリット

- やることを箇条書きで伝えるだけで**優先順位**を付けてくれる
- 大きなタスクを**小さなステップ**に分解してくれる
- 期限や所要時間の**見積もり**もしてくれる
- 毎朝の「今日やること」を整理する習慣が作れる

## 基本の使い方

### 1. やることリストを整理してもらう

頭の中にあるタスクを、とりあえず全部書き出してChatGPTに渡します。

```
以下のタスクを優先度順に整理してください。
それぞれに所要時間の目安もつけてください。

- 来週の会議の資料作成
- 経費精算
- A社への見積もり送付（明日締切）
- チームメンバーの勤怠確認
- 新しいプロジェクトの企画書ドラフト
- メールの返信（5件たまっている）
- 部長への進捗報告
```

ChatGPTの回答例：

```
【優先度：高（今日中）】
1. A社への見積もり送付（明日締切）― 約30分
2. メールの返信（5件）― 約20分
3. 部長への進捗報告 ― 約15分

【優先度：中（今週中）】
4. 来週の会議の資料作成 ― 約2時間
5. チームメンバーの勤怠確認 ― 約15分
6. 経費精算 ― 約30分

【優先度：低（来週でもOK）】
7. 新しいプロジェクトの企画書ドラフト ― 約3時間
```

### 2. 大きなタスクを分解してもらう

「資料作成」のような大きなタスクは、そのままだと手が止まりがちです。

```
「来週の営業会議の資料作成」というタスクを、
すぐ取りかかれる小さなステップに分解してください。
各ステップの所要時間も教えてください。
```

ChatGPTの回答例：

```
1. 前回の会議メモを確認する（5分）
2. 今月の売上データを集める（10分）
3. スライドの構成案を作る（15分）
4. データをグラフにする（15分）
5. スライドに内容を入れる（30分）
6. 上司に確認してもらう（5分）
合計：約1時間20分
```

こうすると「まず前回のメモを見るだけ」から始められるので、取りかかりやすくなります。会議の資料作成が含まれている場合は、[ChatGPTで報告書・レポートを爆速で書く方法](/posts/chatgpt-report-writing/)も参考になります。

## 実践プロンプト集

### 毎朝の日次プランニング

```
今日やるべきことを整理してください。

【今日の予定】
- 10:00 チームミーティング
- 14:00 クライアントとの打ち合わせ
- 16:00 1on1

【やること】
- 打ち合わせ用の提案資料を仕上げる
- 昨日の議事録を共有する
- 経費精算（今週中）
- 新人の研修スケジュールを作る

上記を踏まえて、時間帯ごとのスケジュールを提案してください。
```

### 週次レビュー

```
以下は今週やったことリストです。
振り返りと来週の改善点を提案してください。

【完了】
- A社提案書の提出
- 月次レポート作成
- チーム定例の議事録作成

【未完了】
- B社への見積もり作成（途中）
- 社内研修の資料準備（未着手）

【気づき】
- 会議が多くて作業時間が取れなかった
- 見積もりの情報収集に時間がかかった
```

### プロジェクトのタスク洗い出し

```
以下のプロジェクトに必要なタスクを洗い出してください。
担当者の割り振り案と、マイルストーンも提案してください。

【プロジェクト概要】
- 社内の業務マニュアルをWebサイト化する
- 期限：3ヶ月後
- メンバー：3人（PM1名、デザイナー1名、エンジニア1名）
- 現状：Word文書が50ファイルある
```

## もっと便利に使うコツ

### コツ1：毎日同じプロンプトを使う

タスク管理は「習慣化」が大事です。毎朝同じプロンプトを使うことで、日次プランニングが定着します。

ChatGPTの**カスタム指示**に以下を設定しておくと便利です：

```
私は会社員で、タスク管理を効率化したいと思っています。
タスクを伝えたら、優先度・所要時間・時間帯の提案をしてください。
```

> 💡 カスタム指示の設定方法は[こちらの記事](/posts/chatgpt-custom-instructions/)で解説しています。

### コツ2：ChatGPTを「壁打ち相手」にする

タスクの優先順位に迷ったら、ChatGPTに相談してみましょう。

```
以下の2つのタスク、どちらを先にやるべきですか？
理由も教えてください。

A: 明後日締切の企画書（完成度60%）
B: 今日中に返信が必要なメール（5分で終わる）
```

### コツ3：既存のツールと組み合わせる

ChatGPTで整理したタスクを、普段使っているツールにコピペするのがおすすめです。

- **Notion** → テーブルに貼り付け
- **Googleカレンダー** → 時間帯ごとのスケジュールを反映
- **Todoist / Ticktick** → タスクとして登録

ChatGPTに「Notion用のマークダウン形式で出力して」と指示すれば、そのままコピペできます。スケジュール管理をさらに効率化したい方は、[AIでスケジュール管理を自動化する方法](/posts/ai-schedule-management/)もあわせてご覧ください。

## 注意点

- ChatGPTはあなたの仕事の背景を完全には理解できません。最終判断は自分で行いましょう
- 機密性の高いプロジェクト名や顧客名は伏せて入力してください
- タスクの「実行」はあなた自身です。整理してもらったら、すぐ動くのがコツです

## まとめ

- ChatGPTにタスクを伝えるだけで優先順位・所要時間を整理してくれる
- 大きなタスクは小さなステップに分解してもらうと取りかかりやすい
- 毎朝の日次プランニングを習慣化すると効果的
- 整理したタスクはNotionやカレンダーにコピペして運用する

## よくある質問（FAQ）

### Q: ChatGPTにタスクを覚えておいてもらえますか？
A: ChatGPTは会話が終わると内容を忘れてしまいます。タスクの保存には、NotionやTodoistなど外部ツールを併用してください。ChatGPTの[カスタム指示機能](/posts/chatgpt-custom-instructions/)を使えば、毎回の前提条件は記憶させられます。

### Q: 毎朝のタスク整理を習慣化するコツはありますか？
A: 毎朝同じプロンプトを使うのが効果的です。プロンプトをメモアプリに保存しておき、コピペするだけで日次プランニングが完了する仕組みを作りましょう。

### Q: チームのタスク管理にも使えますか？
A: はい、メンバーのタスクをまとめてChatGPTに渡せば、チーム全体の優先順位付けや担当者の割り振りを提案してくれます。ただし、機密性の高いプロジェクト名は伏せて入力してください。

### Q: タスクの所要時間の見積もりは正確ですか？
A: ChatGPTの見積もりはあくまで目安です。自分の作業スピードに合わせて調整してください。何度か使ううちに、ChatGPTの見積もりと実際の時間のズレが分かってきます。
