## この記事で解決すること

「100ページのPDFを読んで内容を把握したいけど、時間がない」

2026年2月にリリースされたClaude Opus 4.6は、100万トークンのコンテキストウィンドウを搭載しています。これは日本語で約50万文字、つまり新書3〜4冊分の文章を一度に読み込めるということです。

コンテキストウィンドウとは、AIが一度に読める文章の量のことです。

## 他のAIとの比較

| AI | コンテキストウィンドウ | 日本語換算 |
|---|---|---|
| Claude Opus 4.6 | 100万トークン | 約50万文字 |
| ChatGPT（GPT-4o） | 12.8万トークン | 約6万文字 |
| Gemini 3 Pro | 100万トークン | 約50万文字 |

Claudeは長文処理の精度が特に高いと評価されています。

## 使い方

### ステップ1: PDFをアップロードする

1. https://claude.ai にアクセスしてログイン
2. チャット画面の左下にあるクリップアイコン（📎）をクリック
3. PDFファイルを選択してアップロード

### ステップ2: 指示を出す

PDFがアップロードされたら、やりたいことを伝えます。

#### 全体の要約

```
このPDFの内容を、以下の形式で要約してください。

■ 全体の概要（5行以内）
■ 各章のポイント（箇条書き）
■ 重要な数字・データ
■ 結論
```

#### 特定の情報を探す

```
この契約書の中で、解約条件に関する記述をすべて抜き出してください。
```

#### 比較分析

```
この2つのPDFの内容を比較して、違いをまとめてください。
```

2つのPDFを同時にアップロードして比較することもできます。

## 実用的な使い方

### 契約書のチェック

```
この契約書を読んで、以下の観点でリスクがないか確認してください。
- 自動更新条項
- 解約時の違約金
- 知的財産権の帰属
- 損害賠償の上限
```

### 論文の要約

```
この論文を読んで、以下を教えてください。
- 研究の目的
- 使った手法
- 主な結果
- 限界と今後の課題
```

### 議事録の束を分析

```
過去3ヶ月分の議事録をアップロードしました。
この期間で繰り返し議論されているテーマと、
まだ解決していない課題をまとめてください。
```

## 無料プランでの制限

Claudeの無料プランでも長文の読み込みは可能ですが、1日の使用回数に制限があります。大量のPDFを処理したい場合は、有料プラン（Pro：月20ドル）を検討してください。

## 注意点

- 機密文書をアップロードする場合は、社内のAI利用ポリシーを確認してください
- Claudeの回答は参考情報です。契約書の最終判断は専門家に相談してください
- PDFの画像部分（スキャンされた文書など）は読み取れない場合があります

## まとめ

- Claude Opus 4.6は100万トークン（約50万文字）を一度に読める
- PDFをアップロードして指示を出すだけ
- 契約書チェック、論文要約、議事録分析など実用的な使い方が多い
- 長文処理の精度はClaude が現時点で最も高い
