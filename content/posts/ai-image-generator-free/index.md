## 1. ChatGPT（DALL-E）

ChatGPTの中で画像生成ができます。

```
猫がパソコンで仕事をしているイラストを描いて
```

と打つだけで画像が生成されます。

- 無料プランでも使える（回数制限あり）
- 日本語で指示できる
- 会話の流れで「もう少し明るく」など修正指示も可能

すでにChatGPTを使っているなら、一番手軽です。ChatGPTをまだ使ったことがない方は[ChatGPTの始め方ガイド](/posts/chatgpt-first-step/)を参考にしてください。

## 2. Microsoft Copilot（旧Bing Image Creator）

https://copilot.microsoft.com にアクセスして、画像生成を依頼できます。

- Microsoftアカウントがあれば無料
- DALL-E 3ベースで高品質
- 日本語対応

Microsoftアカウントは、Windowsを使っている人ならすでに持っている場合が多いです。

## 3. Canva（AI画像生成機能）

デザインツールのCanvaにもAI画像生成機能があります。

- https://www.canva.com で無料アカウントを作成
- 「マジック生成」機能で画像を作れる
- 生成した画像をそのままデザインに使える

ブログのアイキャッチ画像やSNS投稿用の画像を作るのに便利です。SNS投稿の文章作成には[ChatGPTでSNS投稿を作成する方法](/posts/chatgpt-sns-post/)も活用できます。

## 4. Leonardo.ai

https://leonardo.ai はクリエイター向けのAI画像生成サービスです。

- 無料プランで1日150トークン（数十枚程度）
- スタイルの選択肢が豊富
- 画像の品質が高い

少し操作が複雑ですが、こだわりたい人向け。

## 5. Stable Diffusion（Google Colabで無料実行）

自分でAIモデルを動かして画像を生成する方法です。

- 完全無料（Googleアカウントがあれば使える）
- カスタマイズの自由度が最も高い
- 技術的な知識が少し必要

初心者にはハードルが高いですが、「AIの仕組みを理解したい」人にはおすすめです。AI画像を使った副業に興味がある方は[AIを使った副業の始め方](/posts/ai-side-job-beginner/)もあわせてご覧ください。

## 5つのサービスを全部試して分かったこと

筆者は5つのサービスすべてで同じプロンプト（「夕焼けの海辺でコーヒーを飲む猫」）を試して比較しました。

**所要時間（登録〜最初の画像生成まで）：**
- ChatGPT：2分（既にアカウントあり）
- Microsoft Copilot：3分
- Canva：5分
- Leonardo.ai：8分（UIが英語で少し迷った）
- Stable Diffusion（Colab）：25分（環境構築が必要）

**良かった点：**
- ChatGPTは会話の流れで「もう少し明るく」と修正できるのが直感的
- Microsoft Copilotは画質が一番安定していた
- Leonardo.aiはスタイルの選択肢が豊富で、こだわりたい人向け

**イマイチだった点：**
- Canvaの生成画像は他と比べるとやや平凡な印象
- Stable Diffusionは環境構築のハードルが高く、初心者には厳しい

**結論：** 初心者はChatGPTかMicrosoft Copilotで十分。こだわりたくなったらLeonardo.aiにステップアップするのが自然な流れ。

## 5つのサービスを実際に使い比べた結果

同じプロンプト「夕焼けの海辺を歩く猫」で5つのサービスを比較しました。

### 品質ランキング（筆者の主観）

1. **DALL-E 3（ChatGPT経由）** — 最も自然で高品質。ただし無料枠は少ない
2. **Adobe Firefly** — 商用利用OKで安心。写真風が得意
3. **Stable Diffusion（Clipdrop）** — カスタマイズ性が高い。アート風が得意
4. **Canva AI** — デザイン素材として使いやすい。テンプレートとの組み合わせが便利
5. **Microsoft Designer** — 無料枠が多い。品質は中程度

### 用途別おすすめ

- ブログのアイキャッチ → Adobe Firefly（商用OK・高品質）
- SNS投稿 → Canva AI（テンプレートと組み合わせやすい）
- 個人利用・遊び → DALL-E 3（品質最高）

## よくある質問（FAQ）

### Q: AI画像生成で作った画像は商用利用できますか？
A: サービスによって規約が異なります。Canvaは商用利用OKですが、ChatGPT（DALL-E）やMicrosoft Copilotは利用規約を確認してください。商用利用する場合は、各サービスの最新の利用規約を必ずチェックしましょう。

### Q: 日本語のプロンプトでも高品質な画像が作れますか？
A: ChatGPTやMicrosoft Copilotは日本語プロンプトに対応しています。ただし、英語の方が細かいニュアンスが伝わりやすい場合もあります。うまくいかないときは[ChatGPTプロンプトテンプレート集](/posts/chatgpt-prompt-template/)を参考に、指示の出し方を工夫してみてください。

### Q: AI生成画像の著作権はどうなりますか？
A: 2026年現在、AI生成画像の著作権は法的にグレーな部分があります。生成した画像をそのまま販売する場合は、プラットフォームの規約と最新の法的動向を確認してください。

### Q: スマホからでもAI画像生成はできますか？
A: できます。ChatGPTアプリやCanvaアプリを使えば、スマホからでも画像生成が可能です。Microsoft Copilotもブラウザからアクセスできます。
