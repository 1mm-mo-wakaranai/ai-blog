## GPT for Sheets and Docsとは

GPT for Sheets and Docsは、Googleスプレッドシート上でChatGPT（OpenAI API）を呼び出せる拡張機能です。セルに関数を入力するだけで、AIによるテキスト生成が可能になります。

通常、ChatGPTを使うにはブラウザで画面を開く必要があります。しかしこの拡張機能を使えば、スプレッドシートから離れずにAI処理を実行できます。

たとえば100件の商品名に対して説明文を一括生成する、といった作業が数分で完了します。[ChatGPTの基本的な使い方](/posts/chatgpt-first-step/)を理解していれば、スムーズに導入できるでしょう。

### GPT for Sheetsでできること

主な機能は以下のとおりです。

- テキスト生成（文章作成、要約、翻訳）
- データの分類・タグ付け
- 情報の抽出・整形
- 一括処理（数百行のデータを一度に処理）

Excelでの活用に興味がある方は、[ChatGPT×Excel連携の記事](/posts/chatgpt-excel/)も参考にしてください。

## インストール方法

GPT for Sheets and Docsの導入手順を説明します。

### 手順1：拡張機能を追加する

1. Googleスプレッドシートを開きます
2. メニューバーの「拡張機能」→「アドオン」→「アドオンを取得」を選択します
3. 検索バーに「GPT for Sheets and Docs」と入力します
4. 表示された拡張機能の「インストール」ボタンをクリックします
5. Googleアカウントへのアクセス許可を承認します

インストール完了後、「拡張機能」メニューに「GPT for Sheets and Docs」が追加されます。

### 手順2：拡張機能を有効化する

1. 「拡張機能」→「GPT for Sheets and Docs」→「Enable GPT functions」を選択します
2. サイドバーが表示されれば有効化完了です

この有効化は、スプレッドシートごとに行う必要があります。新しいシートを作成した際は忘れずに実行してください。

## OpenAI APIキーの取得と設定

GPT for Sheetsを動かすには、OpenAIのAPIキーが必要です。APIキーとは、外部サービスからOpenAIのAIモデルを利用するための認証情報です。

### APIキーの取得手順

1. [OpenAI Platform](https://platform.openai.com/)にアクセスします
2. アカウントを作成、またはログインします
3. 左メニューの「API keys」を選択します
4. 「Create new secret key」をクリックします
5. 表示されたキーをコピーして安全な場所に保存します

APIキーは一度しか表示されません。紛失した場合は再発行が必要です。

### スプレッドシートへの設定

1. 「拡張機能」→「GPT for Sheets and Docs」→「Set API key」を選択します
2. コピーしたAPIキーを貼り付けます
3. 「Save API key」をクリックします

設定が完了すると、GPT関数が使えるようになります。

## =GPT()関数の基本的な使い方

GPT for Sheetsの中核となるのが`=GPT()`関数です。セルにこの関数を入力すると、AIが回答を生成してくれます。

### 基本構文

```
=GPT(プロンプト, [値], [モデル], [最大トークン数])
```

- **プロンプト**：AIへの指示文（必須）
- **値**：処理対象のセル参照（任意）
- **モデル**：使用するモデル名（任意、デフォルトはgpt-4o-mini）
- **最大トークン数**：出力の上限（任意）

### 使用例

```
=GPT("以下の文章を50文字以内で要約してください", A2)
```

この例では、A2セルの内容をAIが要約して結果を返します。

```
=GPT("次の単語を英語に翻訳してください: "&A2)
```

セル参照と文字列を結合して、動的なプロンプトを作ることも可能です。

### モデルの指定

```
=GPT("商品説明を書いてください", A2, "gpt-4o")
```

第3引数でモデルを指定できます。精度を重視する場合はgpt-4oを、コストを抑えたい場合はgpt-4o-miniを選択します。

## その他の便利な関数

GPT for Sheetsには、用途に応じた専用関数が用意されています。

### =GPT_LIST()：リスト形式で出力

```
=GPT_LIST("日本の人気観光地を5つ挙げてください")
```

結果を複数のセルに分けて出力します。箇条書きのデータを生成したいときに便利です。

### =GPT_TABLE()：表形式で出力

```
=GPT_TABLE("主要なプログラミング言語5つについて、名前・用途・難易度を表にしてください")
```

複数列×複数行の表形式でデータを生成します。比較表やリサーチ結果の整理に活用できます。

### =GPT_FILL()：パターンを学習して補完

```
=GPT_FILL(A1:B3, A4:A10)
```

既存のデータパターンを学習し、空欄を自動補完します。たとえばA列に商品名、B列に説明文が3行分あれば、残りの行も同じパターンで生成してくれます。

### =GPT_CLASSIFY()：分類・タグ付け

```
=GPT_CLASSIFY(A2, "ポジティブ, ネガティブ, 中立")
```

テキストを指定したカテゴリに分類します。レビュー分析やアンケート集計に役立ちます。

### =GPT_EXTRACT()：情報の抽出

```
=GPT_EXTRACT(A2, "メールアドレス")
```

テキストから特定の情報を抜き出します。非構造化データの整理に使えます。

## 具体的な活用例5選

ここからは、実務で使える具体的なユースケースを紹介します。

### 活用例1：商品説明文の一括生成

ECサイトの商品登録で、説明文の作成に時間がかかっていませんか。GPT for Sheetsを使えば、商品名と特徴から説明文を一括生成できます。

**設定例：**

| A列（商品名） | B列（特徴） | C列（関数） |
|---|---|---|
| ワイヤレスイヤホン | ノイズキャンセリング対応 | =GPT("以下の商品の魅力的な説明文を100文字で書いてください。商品名:"&A2&" 特徴:"&B2) |

数百件の商品でも、関数をコピーするだけで一括処理が可能です。

### 活用例2：メールの下書き作成

定型的なビジネスメールの下書きを自動生成できます。

```
=GPT("以下の要件でビジネスメールの下書きを作成してください。宛先:"&A2&" 用件:"&B2&" トーン:丁寧")
```

件名・宛先・用件をセルに入力しておけば、メール文面が自動で生成されます。[ChatGPTを使ったデータ分析](/posts/chatgpt-data-analysis/)と組み合わせれば、顧客データに基づいたパーソナライズも実現できます。

### 活用例3：データの分類・タグ付け

顧客からの問い合わせやレビューを自動分類する例です。

```
=GPT_CLASSIFY(A2, "製品に関する質問, 配送について, クレーム, その他")
```

手動で分類していた作業を自動化できます。数千件のデータでも一貫した基準で分類が可能です。

### 活用例4：翻訳

多言語対応が必要な場面で、一括翻訳ができます。

```
=GPT("以下の日本語を英語に翻訳してください。自然な表現にしてください: "&A2)
```

Google翻訳との違いは、文脈やトーンを指示できる点です。「カジュアルに」「ビジネス向けに」といった指定が可能です。

### 活用例5：要約

長文のテキストを指定した文字数に要約します。

```
=GPT("以下の文章を3行で要約してください: "&A2)
```

議事録の要約、記事のダイジェスト作成、レポートの概要生成など、幅広い場面で活用できます。

## 料金の目安（APIトークン消費）

GPT for Sheetsの利用には、OpenAI APIの従量課金が発生します。拡張機能自体は無料ですが、API呼び出しごとにトークンが消費されます。

### トークンとは

トークンはAIが処理するテキストの単位です。日本語の場合、1文字あたり約1〜3トークンを消費します。英語では1単語が約1トークンです。

### 料金の目安（2026年5月時点）

| モデル | 入力（100万トークンあたり） | 出力（100万トークンあたり） |
|---|---|---|
| gpt-4o-mini | $0.15 | $0.60 |
| gpt-4o | $2.50 | $10.00 |

**具体的なコスト例：**

- 100件の商品説明文生成（gpt-4o-mini）：約$0.01〜$0.05
- 1,000件のデータ分類（gpt-4o-mini）：約$0.05〜$0.20
- 100件の長文要約（gpt-4o）：約$0.50〜$2.00

日常的な業務利用であれば、月額数百円〜数千円程度に収まるケースが多いです。

### コストを抑えるコツ

- gpt-4o-miniを優先的に使う（精度が十分な場合）
- 最大トークン数を指定して出力を制限する
- 不要な再計算を防ぐため、結果を「値のみ貼り付け」で固定する

## 注意点とベストプラクティス

GPT for Sheetsを安全に使うために、いくつかの注意点があります。

### レート制限について

OpenAI APIにはレート制限（一定時間内のリクエスト上限）があります。無料アカウントや新規アカウントでは制限が厳しく設定されています。

- 大量のセルに一度に関数を入力すると、エラーが発生する場合があります
- 50〜100件ずつ段階的に処理することを推奨します
- エラーが出た場合は数分待ってから再実行してください

### 個人情報の取り扱い

APIに送信したデータは、OpenAIのサーバーで処理されます。以下の点に注意してください。

- 個人情報（氏名、住所、電話番号など）を含むデータの送信は避ける
- 社内の機密情報を処理する場合は、社内ポリシーを確認する
- OpenAIのデータ利用ポリシーを事前に確認する
- API経由のデータはモデルの学習には使用されません（2026年5月時点）

### その他の注意点

- シートを開くたびに関数が再計算される場合があります（トークン消費に注意）
- 結果を確定したら「値のみ貼り付け」で固定しましょう
- APIキーは他人と共有しないでください
- 月額の利用上限（Usage limits）をOpenAIダッシュボードで設定しておくと安心です

## よくある質問（FAQ）

### Q1. GPT for Sheetsは無料で使えますか？

拡張機能のインストール自体は無料です。ただし、AIの処理にはOpenAI APIの従量課金が発生します。新規アカウントには無料クレジットが付与される場合がありますが、継続利用には支払い方法の登録が必要です。

### Q2. 関数を入力してもエラーが出ます。原因は何ですか？

主な原因は以下の3つです。

1. APIキーが正しく設定されていない
2. OpenAIアカウントの残高が不足している
3. レート制限に達している

「拡張機能」→「GPT for Sheets and Docs」→「Enable GPT functions」を再実行することで解決する場合もあります。

### Q3. 日本語でも正確に動作しますか？

はい、日本語に対応しています。ただし英語と比較すると、同じ内容でもトークン消費量が多くなる傾向があります。プロンプトを日本語で書いても問題なく動作します。

### Q4. 処理結果がセルを開くたびに変わってしまいます。固定する方法はありますか？

結果が確定したら、対象セルをコピーして「編集」→「特殊貼り付け」→「値のみ貼り付け」を実行してください。これにより関数が結果の値に置き換わり、再計算されなくなります。

### Q5. 一度に何件まで処理できますか？

技術的な上限は明確に定められていませんが、レート制限の関係で一度に数百件程度が現実的です。大量データを処理する場合は、50〜100件ずつ段階的に実行することを推奨します。
