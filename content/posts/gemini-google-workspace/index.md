## この記事で解決すること

「GeminiってGoogleのサービスと連携できるらしいけど、具体的に何ができるの？」

2026年のアップデートで、GeminiはGmail・Googleドキュメント・カレンダー・フォト・YouTubeなど、Googleのサービスと直接つながるようになりました。無料プランでも使える機能を紹介します。

## できること一覧

| 連携先 | できること |
|---|---|
| Gmail | メールの要約、返信の下書き、特定のメールを探す |
| Googleドキュメント | 文章の作成・要約・校正 |
| Googleカレンダー | 予定の確認・追加 |
| Googleフォト | 写真の検索（「去年の旅行の写真」など） |
| YouTube | 動画の内容を要約 |

## 設定方法

1. https://gemini.google.com にアクセスしてログイン
2. 右上の設定アイコン → 「拡張機能」を開く
3. 使いたいサービス（Gmail、ドキュメントなど）をオンにする

これだけで連携完了です。

## 実際に使ってみる

### Gmailと連携：メールを探す

```
先週、田中さんから届いたメールの内容を教えて
```

Geminiが自分のGmailを検索して、該当するメールの内容を要約してくれます。

### Gmailと連携：返信を作る

```
田中さんからの見積もり依頼メールに、
「確認しました。来週中に回答します」という趣旨で返信を作って
```

### カレンダーと連携：予定を確認

```
来週の予定を教えて
```

Googleカレンダーの内容を読み取って、一覧で教えてくれます。

### カレンダーと連携：予定を追加

```
4月25日の14時から1時間、「企画会議」をカレンダーに追加して
```

### YouTubeと連携：動画の要約

```
この動画の内容を要約して
https://www.youtube.com/watch?v=xxxxx
```

長い動画を見る時間がないときに便利です。

## ChatGPTにはできないこと

この「Googleサービスとの直接連携」はGeminiだけの強みです。ChatGPTでは自分のGmailを検索したり、カレンダーに予定を追加したりはできません。

Googleのサービスを普段から使っている人にとっては、Geminiの方が実用的な場面が多いです。

## 注意点

- 連携をオンにすると、GeminiがGmailやカレンダーの内容を読み取ります
- 機密性の高い情報がある場合は、連携するサービスを選んでオンにしてください
- 連携はいつでもオフにできます

## まとめ

- GeminiはGoogleサービスと直接連携できる唯一のAI
- Gmail、カレンダー、ドキュメント、フォト、YouTubeに対応
- 設定は「拡張機能」をオンにするだけ
- 無料プランでも使える
