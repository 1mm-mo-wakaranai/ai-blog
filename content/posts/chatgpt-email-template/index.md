ビジネスメールを書くのに毎回悩んでいませんか？

ChatGPTを使えば、場面に合ったビジネスメールが**一瞬**で完成します。この記事では、基本のプロンプト構成と、場面別のテンプレートを紹介します。

## 基本のプロンプト構成

ビジネスメールを作成するときは、以下の4つの要素をプロンプトに含めましょう。

```text
以下の条件でビジネスメールを作成してください。

- 宛先：【相手の立場・関係性】
- 目的：【メールの目的】
- トーン：【丁寧 / ややカジュアル / フォーマル】
- 補足：【追加で伝えたいこと】
```

この4要素を押さえるだけで、適切なトーンのメールが生成されます。

## 場面別テンプレート5選

### 1. お礼メール

```text
以下の条件でビジネスメールを作成してください。

- 宛先：取引先の担当者（山田様）
- 目的：先日の打ち合わせのお礼と、次回の打ち合わせ日程の確認
- トーン：丁寧
- 補足：打ち合わせでは新商品の提案をいただいた。非常に参考になった。次回は来週水曜を希望。
```

### 2. お詫びメール

```text
以下の条件でビジネスメールを作成してください。

- 宛先：取引先の部長（鈴木部長）
- 目的：納品遅延のお詫びと今後の対応策の説明
- トーン：フォーマル（深くお詫びする内容）
- 補足：納品が3日遅れる見込み。原因は製造工程のトラブル。代替案として一部先行納品が可能。
```

### 3. 日程調整メール

```text
以下の条件でビジネスメールを作成してください。

- 宛先：社内の他部署マネージャー（佐藤さん）
- 目的：来週のプロジェクト会議の日程調整
- トーン：ややカジュアル（社内メール）
- 補足：候補日は来週の火曜・水曜・木曜の午後。所要時間は1時間。オンライン会議希望。
```

### 4. リマインドメール

```text
以下の条件でビジネスメールを作成してください。

- 宛先：チームメンバー全員
- 目的：金曜日が締め切りの月次レポート提出のリマインド
- トーン：ややカジュアル（チーム内連絡）
- 補足：提出先は共有フォルダ。テンプレートは先月と同じ。不明点があれば気軽に聞いてほしい。
```

### 5. お断りメール

```text
以下の条件でビジネスメールを作成してください。

- 宛先：営業担当者（田中様）
- 目的：提案いただいたサービスの導入を見送る旨の連絡
- トーン：丁寧（相手の気持ちに配慮）
- 補足：提案内容は魅力的だったが、今期の予算の都合で見送り。来期に改めて検討したい。
```

## 応用テクニック

### 受信メールへの返信を作成する

受信したメールの内容をChatGPTに貼り付けて、返信を作成してもらうこともできます。

```text
以下のメールに対する返信を作成してください。
トーンは丁寧で、提案を前向きに検討する内容にしてください。

---
【受信メールの内容をここに貼り付け】
```

### 英語メールの作成

日本語で指示を出して、英語のビジネスメールを作成することもできます。

```text
以下の条件で英語のビジネスメールを作成してください。
日本語で内容を説明するので、自然なビジネス英語に翻訳してください。

- 宛先：海外の取引先（Mr. Johnson）
- 目的：新製品のサンプル送付の連絡
- トーン：Professional but friendly
- 補足：サンプルは来週月曜に発送予定。届いたらフィードバックをいただきたい。
```

## メール作成のコツ

### トーンの指定を具体的にする

「丁寧に」だけでなく、より具体的に指定すると精度が上がります。

- 「新入社員が上司に送るような丁寧なトーンで」
- 「長年の取引先に送る、親しみのあるフォーマルなトーンで」
- 「社内のSlackに投稿するようなカジュアルなトーンで」

### 送信前に必ず確認する

⚠️ ChatGPTが生成したメールは、**必ず送信前に自分で確認**してください。

チェックポイント：
- **事実関係は正しいか**（日付、金額、名前など）
- **トーンは適切か**（相手との関係性に合っているか）
- **自分の言葉として自然か**（違和感がある表現は修正する）
- **機密情報が含まれていないか**

AIが生成した文章をそのまま送るのではなく、自分のフィルターを通すことが大切です。

## まとめ

ChatGPTを使えば、ビジネスメールの作成時間を大幅に短縮できます。

ポイントは3つです：

1. **宛先・目的・トーン・補足**の4要素をプロンプトに含める
2. **場面別のテンプレート**を活用して効率化する
3. **送信前に必ず自分で確認**する

まずは今日のメールから試してみてください。
